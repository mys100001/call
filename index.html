服务器部署指南

如需自建信令服务器，请按照以下步骤操作：

信令服务器 (Node.js + WebSocket)

  1. 创建一个新目录，并在其中创建 server.js 文件
  2. 复制以下服务器代码到文件中：
// server.js - WebRTC信令服务器
const WebSocket = require('ws');

// 创建WebSocket服务器，监听8080端口
const wss = new WebSocket.Server({ port: 8080 });
const users = {};

// 处理客户端连接
wss.on('connection', (ws) => {
  let userId = null;
  
  // 处理客户端消息
  ws.on('message', (message) => {
    const data = JSON.parse(message);
    
    switch (data.type) {
      case 'register':
        // 用户注册
        userId = data.userId;
        users[userId] = ws;
        console.log(`用户注册: ${userId}`);
        
        // 通知用户注册成功
        ws.send(JSON.stringify({
          type: 'registered',
          userId: userId
        }));
        
        // 发送在线用户列表
        broadcastUserList();
        break;
        
      case 'offer':
        // 发送offer给被呼叫方
        console.log(`Offer from ${data.from} to ${data.to}`);
        if (users[data.to]) {
          users[data.to].send(JSON.stringify({
            type: 'offer',
            from: data.from,
            offer: data.offer
          }));
        }
        break;
        
      case 'answer':
        // 发送answer给呼叫方
        console.log(`Answer from ${data.from} to ${data.to}`);
        if (users[data.to]) {
          users[data.to].send(JSON.stringify({
            type: 'answer',
            from: data.from,
            answer: data.answer
          }));
        }
        break;
        
      case 'candidate':
        // 转发ICE候选
        if (users[data.to]) {
          users[data.to].send(JSON.stringify({
            type: 'candidate',
            from: data.from,
            candidate: data.candidate
          }));
        }
        break;
        
      case 'callEnd':
        // 转发通话结束消息
        if (users[data.to]) {
          users[data.to].send(JSON.stringify({
            type: 'callEnd',
            from: data.from
          }));
        }
        break;
    }
  });
  
  // 处理连接关闭
  ws.on('close', () => {
    if (userId) {
      delete users[userId];
      console.log(`用户断开连接: ${userId}`);
      broadcastUserList();
    }
  });
  
  // 广播在线用户列表给所有用户
  function broadcastUserList() {
    const userList = Object.keys(users);
    const message = JSON.stringify({
      type: 'userList',
      users: userList
    });
    
    wss.clients.forEach(client => {
      if (client.readyState === WebSocket.OPEN) {
        client.send(message);
      }
    });
  }
});

console.log('信令服务器运行在 ws://localhost:8080');
  1. 安装依赖：npm install ws
  2. 运行服务器：node server.js
  3. 将前端页面中的服务器地址改为 ws://你的服务器IP:8080

通过Glitch免费部署（推荐）

  1. 访问 glitch.com 并注册账号
  2. 点击 "New Project" → "Import from GitHub"
  3. 使用GitHub仓库: https://github.com/yourusername/webrtc-signaling-server (需先上传上述代码到GitHub)
  4. 或直接在Glitch中创建新项目，将上面的服务器代码复制到server.js
  5. 在 package.json 中添加依赖："dependencies": { "ws": "^8.0.0" }
  6. Glitch会自动部署，记下项目URL，将前端页面中的服务器地址改为 wss://你的项目名.glitch.me

使用说明

  1. 输入您的用户名（可选，默认自动生成）
  2. 输入对方的用户名（对方也需要打开此页面并注册用户名）
  3. 选择音频或视频通话模式
  4. 点击电话按钮开始呼叫
  5. 对方接听后即可开始通话
  6. 点击麦克风或摄像头图标可切换开启/关闭
  7. 通话结束后点击挂断按钮
  8. 如需自建服务器，请按照上方服务器部署指南操作
